My Computer

Yogscast Jingle Jam 2020 Unity

A frustration based game, the theme "surprise" was turned into annoying pop ups. The player controls the bouncy hand to answer the 3 emails in their inbox, all the time avoiding the pop ups that appear surprisingly. My Computer came in 14th in the voting.

Quinn was the sole developer on the project and took this game jam to focus on creating a complete extendable system that allowed for different types of pop up to appear, as well as creating a fun but challenging input system for the player.
